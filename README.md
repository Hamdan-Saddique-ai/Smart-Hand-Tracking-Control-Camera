# 🎥 Smart Hand Tracking & Control Camera

A real-time computer vision system that enables touchless interaction using hand gestures. Built with OpenCV and MediaPipe, this project allows users to control different camera modes and visual effects using only their hands.

---

## 🚀 Features

* ✋ Real-time hand tracking and detection
* 🤖 Multiple gesture recognition (10+ gestures)
* 🎮 Interactive modes (DJ, Gun, Peace, Fight, etc.)
* 🎨 Dynamic visual effects and overlays
* 📸 Capture and save sketch images
* 🔄 Motion-based gesture detection (Yo-Yo movement)

---

## 🖐️ Supported Gestures

| Gesture         | Action      |
| --------------- | ----------- |
| 🔫 Gun Sign     | Gun Mode    |
| ✌️ Peace Sign   | Peace Mode  |
| 🎧 DJ Sign      | DJ Mode     |
| ✊ Fist          | Fight Mode  |
| 🖐️ Open Hand   | Normal Mode |
| 👍 Thumbs Up    | Like Mode   |
| 👌 OK Sign      | OK Mode     |
| ✋ Stop Sign     | Stop Mode   |
| ☝️ Pointing     | Point Mode  |
| 🔄 Yo-Yo Motion | Music Mode  |
